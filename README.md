# Printmode Tools

React application for working with printmodes.

# Getting Started

Application requires `NodeJS`.

## Docker

If Docker is installed:

```
./build-container
./launch-container
```

## VS Code with `Remote - Containers` Extension

If the extension `Remote - Containers` is enabled in VS Code:

```
./build-container
code .
```

Reopen VS Code inside the container.

# Usage

To start the application:

```
yarn install
yarn start
```

Now you can visit [`localhost:1234`](http://localhost:1234) from your browser.
